## 📸 QC decision board

<table>
  <tr>
    <td align="center"><strong>PASS</strong><br><sub>All key angles are clean and details match.</sub></td>
    <td align="center"><strong>RETAKE</strong><br><sub>Critical proof is blurry, cropped, dark, or missing.</sub></td>
    <td align="center"><strong>HOLD</strong><br><sub>Evidence conflicts with size, label, color, or shape.</sub></td>
    <td align="center"><strong>SKIP</strong><br><sub>The flaw is not worth international shipping.</sub></td>
  </tr>
</table>
