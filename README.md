## ✅ Photo proof checklist

- [ ] Front, side, tag, detail, and measurement photos are clear.
- [ ] Key category details match the expected product proof.
- [ ] Labels, stitching, print, hardware, or shape do not show obvious red flags.
- [ ] Size evidence supports the item you meant to order.
- [ ] Retakes are requested before the item enters a parcel.
- [ ] Final parcel weight, packaging, and shipping risk still make sense.
